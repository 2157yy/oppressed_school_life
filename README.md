# 反讽教育的恐怖游戏

### 1、随说
- 这是一个做梦梦见的游戏，当时我玩的很害怕，太真实了
- 由于本人认为讽刺教育也许是个不错的题材，所以醒来立刻动手来写这个文档了
- 我希望这个作品能够很好的完成，虽然这个方向我从来没有研究过，但是只要开始，我相信一切难题都会慢慢解决

### 2.设想
- 第一人称游戏
- 场景昏暗
- 追逐者有手电，角色如电锯人、飞蛾怪、保安
- 玩家初始无武器，只能依靠跑和躲避
- 关卡设置如拉闸等操作
- 机制：
  1. 没有固定安全屋，每次死亡后，安全屋会刷新位置，即躲在哪，追逐者都可能会开门（这里可能需要一个随机算法）
  2. 追逐者设定知道玩家躲藏位置，挡玩家藏起来的时候，追逐者可在门外长时间逗留，以达到恐怖目的（需设定随机频率算法，以决定哪次会在玩家门口逗留）

- 这是一个完整的故事，倒叙讲解，慢慢的真相浮现
